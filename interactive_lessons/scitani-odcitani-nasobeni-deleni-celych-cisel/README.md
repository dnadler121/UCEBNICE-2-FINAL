# Interaktivní lekce pro UČEBNICI 2.0

Zařazení:

- Předmět: Matematika
- Škola: Střední škola
- Ročník: 1. ročník
- Téma: Základní početní operace
- Lekce: Sčítání, odčítání, násobení a dělení celých čísel

Balíček se importuje v učitelské administraci tlačítkem **Import aplikace**.

Obsahuje povinný `lesson.json`. Student dokončí 10 příkladů a výsledek,
známka i datum dokončení se uloží do databáze UČEBNICE 2.0.
