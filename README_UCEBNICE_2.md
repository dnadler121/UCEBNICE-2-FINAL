# UČEBNICE 2.0 – tato aktualizace

Tato verze zachovává současnou biologii/občanku a přidává:

- krásnou hlavní stránku se třemi vstupy,
- katalog Biologie + Občanská výchova,
- samostatné katalogy Matematika a Informatika,
- automatické řazení podle `Předmět → Škola a ročník → Téma → Lekce`,
- povinné zařazení při vytváření každé HTML lekce,
- mazání jednoho nebo všech výsledků bez smazání studentů a průběžného pokroku.

## Jak vytvořit lekci

V editoru vždy vyplň:

1. Předmět
2. Škola a ročník
3. Téma
4. Název lekce
5. HTML výklad a otázky

Nová škola nebo ročník nemusí být předem založený. Stačí napsat nový název, například `Gymnázium – 2. ročník`. Aplikace vytvoří nové zařazení automaticky.

## Matematika a informatika

Katalogy jsou připravené. Interaktivní import samostatných Flask podaplikací bude další krok; tato verze zatím bezpečně připravuje navigaci, databázi a pravidla zařazení.
