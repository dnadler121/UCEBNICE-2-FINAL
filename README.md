# Montessori Engine V1.1

## Spuštění

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Otevři: http://127.0.0.1:5000

## Co je nového

- Zachovaný původní studentský design.
- Registrace studentů učitelem.
- Přehled databáze: aktuální téma studenta a poslední test.
- Mazání studentů včetně výsledků.
- Výrobna otázek v editoru lekce: výběr odpovědi a krátká odpověď.
- Otázky zůstávají ve studentské části po jedné a náhodně.
- Závěrečný test má výklad vedle.
