# Montessori Engine – PDF výklad

V této verzi zůstává vše ostatní stejné, mění se jen výklad:

- výklad se vkládá jako `.pdf`,
- v editoru se po výběru PDF zobrazí náhled,
- student vidí PDF přímo ve výkladu,
- otázky, aktivita, přihlášení, studenti i design zůstávají beze změny.

## Instalace

```bash
pip install -r requirements.txt
python app.py
```

## Doporučený postup

1. Připrav výklad ve Wordu nebo LibreOffice.
2. Vlož do něj obrázky, tabulky a popisky.
3. Exportuj jako PDF.
4. V editoru lekce vyber PDF v části „Import PDF výkladu“.
5. Ulož lekci.
